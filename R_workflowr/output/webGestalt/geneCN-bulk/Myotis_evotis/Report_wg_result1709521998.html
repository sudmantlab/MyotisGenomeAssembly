Job summary

  • Enrichment method: ORA
  • Organism: hsapiens
  • Enrichment Categories: pathway_Reactome
  • Interesting list: geneCN-Myotis_evotis_1709521998.txt. ID type: genesymbol
  • The interesting list contains 1200 user IDs in which 1176 user IDs are unambiguously mapped to 1176 unique entrezgene IDs and 24 user IDs can not be mapped to any entrezgene ID.
  • The GO Slim summary are based upon the 1176 unique entrezgene IDs.
  • Among 1176 unique entrezgene IDs, 764 IDs are annotated to the selected functional categories and also in the reference list, which are used for the enrichment analysis.
  • Reference list: uploads/geneCN-allGenes_1709521998.txt ID type: genesymbol
  • The reference list can be mapped to 16337 entrezgene IDs and 9129 IDs are annotated to the selected functional categories that are used as the reference for the enrichment analysis.

Parameters for the enrichment analysis:

  • Minimum number of IDs in the category: 5
  • Maximum number of IDs in the category: 2000
  • FDR Method: BH
  • Significance Level: FDR < 1

Based on the above parameters, 224 categories are identified as enriched categories, in which 100 most significant categories and represetatives in the reduced sets are shown in this report. All significant categories can be downloaded from the 'Result Download' link.
